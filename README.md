# 🎓 Student Hub Application

## 📌 Overview

The Student Hub is a web based application designed to improve student access to essential university services in one central platform. The app provides real-time information and easy navigation for key features such as the gym, library, student union events, and campus transport.

---

## 🎯 Project Aim

The aim of this project is to design and develop a user-friendly application that enhances student experience by:

* Centralising important services
* Providing real time updates (e.g. gym capacity, bus times)
* Improving accessibility and usability

---

## 🚀 Features

### 🏠 Home Dashboard

* Welcome interface for students
* Search functionality for quick access
* Quick-access cards:

  * Gym
  * Travel
  * Events
  * Library

### 🏋️ Gym

* Live capacity tracking
* Daily class schedule
* Booking system
* Opening hours

### 🚌 Travel & Transport

* Campus locations and directions
* Interactive map
* Transport information

### 📚 Library

* Study environment status (e.g. quiet levels)
* Location details

### 📰 News & Media

* Updates and announcements
* Student-related content
